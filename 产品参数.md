# 产品参数

## ESP32主控参数

| **名称**     | **参数**                                             |
|--------------|------------------------------------------------------|
| **模组**     | ESP32-32E                                            |
| **CPU**      | ESP32-D0WD-V3,Xtensa双核32位LX6 微处理器             |
| **主频**     | 240MHz（最大）                                       |
| **存储**     | 448KB ROM+520KB SRAM+16KB RTC SRAM+4MB外接QSPI Flash |
| **WIFI**     | 2.4GHz、802.11b/g/n模式                              |
| **蓝牙**     | 蓝牙 V4.2 BR/EDR 和蓝牙LE标准                        |
| **工作电压** | 3.0\~3.6(V)                                          |
| **工作温度** | –40\~85(℃)                                           |
